# Pasos para ejecutar la aplicación:
### 1) Descargar el repositorio
### 2) Ejecutar "npm install"
### 3) Ejecutar "npm run dev"

# Pasos para salida a producción de la aplicación:
### 1) Descargar el repositorio
### 2) Ejecutar "npm install"
### 3) Ejecutar "npm run build"
### 4) Ejecutar "npm start"
